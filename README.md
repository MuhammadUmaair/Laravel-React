# Getting Started with Create Laravel 8 App

```
composer create-project laravel/laravel:^8.0 laravel-api
 
cd example-app
 
php artisan serve
```

### Hierarchy
```

📂 app
    📂 Http
        📂 Controllers
        📂 Middleware
    📂 Models
        🗏 Dashboard.js
        🗏 Home.js
        🗏 Profile.js
📂 routes
    🗏 Routes.js

```
